5.3 集合的表示

  • 枚举集合的子集:
xxxxxxxxxx
for(int S = 0; S < 1 << n; S++) {//对子集的处理}
  • 枚举某个特定集合的子集:
xxxxxxxxxx
int sub = sup;
do {
  //对子集的处理
  sum = (sub - 1) & sup;
} while(sub != sup); //处理完0后, 会有-1 & sup = sup
  • 枚举集合的大小为的子集
xxxxxxxxxx
int comb = (1<<k) - 1;
while(comb < 1<<n) {
  int x = comb & -comb, y = comb + x;
  comb = ((comb & -y) / x >> 1) | y;
}

5.4 BITSET的使用

首先，声明bitset:

xxxxxxxxxx
#include <bitset>
using std::bitset;

初始化：

xxxxxxxxxx
bitset<n> b;
bitset<n> b(unsigned long u);

上述语句声明了一个n位全部为0的bitset，第二个语句用一个unsigned long long变量去初始化bitset。 bitset的更多操作： 

b1 = b2 & b3;//按位与
b1 = b2 | b3;//按位或
b1 = b2 ^ b3;//按位异或
b1 = ~b2;//按位补
b1 = b2 << 3;//移位
